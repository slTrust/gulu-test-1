```
<g-input/>

```

> #### 为什么自闭和不行呢？

因为vue的文档说的很清楚，vue的模版使用的是 html语法，而没说它是 xml语法


```
所以 如果是 xml 你可以<g-input/> 它允许自闭和

但是 html呢？

html语法 规定自定义标签不允许自闭和
所以你应该老老实实的写成

<g-input></g-input>
```

#### 单文件组件里 name属性的作用

- 请先安装 vue-devtools

这个name是用来调试的，方便你定位你的组件，如果你不起名它就是GInput


#### 单文件组件里 scoped的作用

- 样式的隔离，避免组件之间的冲突 - 推荐每个组件都使用它

```
<style scoped></style>
```
